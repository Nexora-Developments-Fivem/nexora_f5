# Nexora F5 Menu

**Multi-Framework Menu System for FiveM**

A powerful, lightweight and fully customizable F5 menu for your FiveM server.

## Features

- **Multi-Framework** - ESX, QBCore, Custom
- **HUD Management** - Universal HUD detection & control
- **Radio System** - Integrated radio with pma-voice
- **Vehicle Control** - Doors, windows, engine management
- **Cinematic Mode** - Immersive filming experience
- **Performance** - FPS boost options included
- **Translations Ready** - Easy localization support

## Installation

1. Download and extract to your resources folder
2. Add `ensure nexora_f5` to your server.cfg
3. Configure here : config/config.lua
4. Import the SQL file located in `nexora.sql` into your server database.
5. Restart your server to apply the changes.

## Default Keybind

Press **F5** to open the menu
